# Illusion Breaker Extension Icons

This folder should contain icon images for the Chrome extension.

Required files:

- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Creating Icons

You can create icons using any image editor. The icons should:

- Have a transparent background
- Use the Illusion Breaker branding (blue background with white "I")
- Be simple and recognizable at small sizes

### Quick Icon Creation

You can use an online tool like:

- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

Or create a simple SVG and convert it to PNG at different sizes.

For demo purposes, you can use placeholder icons or create simple colored squares
with the letter "R" in white text.
